# 監視資料庫

您的所有 Campaign 執行個體資產、工作流程和資料都儲存在資料庫中。資料庫達到其最大容量可能會導致執行個體發生問題（無法登入、無法傳送電子郵件等）。 因此，必須定期監控執行個體的資料庫，以確保最佳效能。
