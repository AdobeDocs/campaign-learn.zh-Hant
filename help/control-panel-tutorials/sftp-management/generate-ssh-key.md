# 產生 SSH 金鑰

## 存取控制面板 SFTP 管理

若要存取「控制面板」的「SFTP 管理」，請前往：

* [Experience Cloud 首頁](https://experience.adobe.com/#/home) > 解決方案選擇器：**[!DNL Campaign]** > **控制面板** 卡片 > **SFTP 卡**

   或

* 直接從 URL：[http://experience.adobe.com/#/controlpanel/sftp](https://experience.adobe.com/)
