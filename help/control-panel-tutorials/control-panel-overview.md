# 控制面板 - 概觀

「控制面板」可以讓 Adobe Campaign 管理員監視關鍵資產並執行管理工作，例如依執行個體管理 SFTP 儲存、允許列出 IP 地址或管理 GPG 金鑰。

查看 [發行說明](https://experienceleague.adobe.com/docs/control-panel/using/release-notes.html?lang=zh-Hant) 有關最新產品版本的資訊。
