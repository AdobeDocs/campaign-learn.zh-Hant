# 控制面板 - 概觀

Adobe Campaign 控制面板可讓 Adobe Campaign 管理員監控關鍵資產並執行管理工作，例如依執行個體管理 SFTP 儲存、管理 GPG 金鑰或子網域和憑證。

## 新增功能

* **[監控資料庫](/help/control-panel-tutorials/performance-monitoring/monitoring-databases.md)**

   *瞭解如何監控執行個體的資料庫使用情況。*
