# 建立工作流程

Adobe Campaign包含工作流程模組，可讓您協調應用程式伺服器不同模組的所有流程和任務。 此全方位的圖形環境可讓您設計流程，包括細分、行銷活動執行、檔案處理、人員參與等。 工作流程引擎會執行並追蹤這些程式。

工作流程會在行銷活動管理程式的不同內容和階段中進行。

此影片說明如何在Adobe Campaign中建立工作流程。
